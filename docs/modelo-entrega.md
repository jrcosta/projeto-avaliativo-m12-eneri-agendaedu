## 7) Roteiro de submissão no AVA

1. Confirmar se o PR final para `main` foi aprovado/mergeado.
2. Verificar se pipeline está verde no commit final.
3. Confirmar que o vídeo está como **não listado** no YouTube.
4. Colar no AVA:
   - link do repositório;
   - link do quadro do GitHub;
   - link do vídeo;
   - (opcional) link do PR final.
5. Registrar data e horário da submissão: `[preencher]`.
